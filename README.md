## 📖 Overview
This is a zero-config CLI tool that crawls a target webpage to harvest external links and identify those returning 4xx or 5xx HTTP errors. It bypasses expensive SaaS subscriptions and complex configurations to provide immediate offensive reconnaissance on competitor content. The tool performs parallel status checks using HEAD requests with GET fallbacks, generating a structured CSV report of dead links including status codes and anchor text. It is designed for professionals who need to map outreach targets and link opportunities without the bloat of traditional auditing platforms.
